<hr>

###Clone Repository
```
  git clone https://github.com/ByungjinKang/trackers.git
```

###build.gradle
```
  ./gradlew --refresh-dependencies
```

###run application
- TrackerApplication.java
```
  ./gradlew bootRun  
```
